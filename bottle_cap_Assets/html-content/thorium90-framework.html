Overview

Thorium90 is a lightweight PHP framework following the MVC (Model-View-Controller) architectural pattern. It provides a structured approach to building web applications with features like database abstraction, security measures, and modular components.

System Requirements

  • PHP 7.4 or higher
  • MySQL 5.7 or higher
  • Apache/Nginx web server
  • mod_rewrite enabled

Required PHP Extensions

  • PDO
  • mysqli
  • session
  • json

Framework Architecture

rad_modules: Core System Functionality

  • Configuration
  • Database connection
  • Security features
  • Helper functions

pip_boy_MVC: MVC Implementation

  • Models (fusion_core_Models)
  • Views (wasteland_Views)
  • Controllers (overlord_Controllers)

bottle_cap_Assets: Frontend Resources

  • CSS
  • Images
  • Custom styling

Installation

  1. Clone the Repository

    git clone https://github.com/your-username/thorium90.app.git

  2. Configure Web Server

    Point your web server to the project directory

  3. Import Database Schema

    mysql -u your_username -p your_database < thorium90-app.sql

  4. Configure Database Connection

    Update settings in rad_modules/_system/config.php

Configuration

Main Configuration File

Located at rad_modules/_system/config.php


// Database Configuration
define("DB_TYPE", "MYSQL");
define("DB_SERVER", "localhost");
define("DB_DATABASE", "your_database");
define("DB_USER", "your_username");
define("DB_PASSWORD", "your_password");

// Application Configuration
define("APP_DIRECTORY", "/");
define("DEFAULT_TITLE", "Your App Title");
define("SESSION_LIFETIME", 3600);
define("INACTIVITY_TIMEOUT", 900);
